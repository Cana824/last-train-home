The Last Train Home

A branching narrative game built with React, TypeScript and CSS.

Players navigate a late-night train station where small decisions lead to different story paths and endings. Choices affect the outcome of the story.

Features
Branching storyline with multiple endings
Choice-based gameplay
Inventory system
Ending tracker / collectible endings
Dynamic scene backgrounds
Interactive UI styled as the front of a train
Restart and replay functionality
Technologies Used
React
TypeScript
Vite
CSS

Your decisions determine whether the night ends normally… or becomes something much stranger.
